**QuizIF** adalah aplikasi kuis berbasis web dengan beberapa level permainan.  
Setiap level berisi kumpulan pertanyaan yang terbagi menjadi dua jenis:
- **Pilihan Ganda (Multiple Choice)**  
- **Isian Singkat (Text Input)**  

Aplikasi ini memiliki sistem level yang **terbuka secara bertahap** â€” pengguna hanya dapat melanjutkan ke level berikutnya jika skor pada level sebelumnya mencukupi.  
Setiap jawaban benar akan menambah **poin skor**, sedangkan jawaban salah tidak menambah poin.
